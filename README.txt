README.md

Code for running analyses described in "Shared understanding is correlated with shared
neural responses." Paper [link](https://www.sciencedirect.com/science/article/pii/S1053811918307948)

ISC
Code for running isc within a group and between groups, as well as conduct phase-randomization for assessing statistical significance.

RSA
Code for running intersubject RSA, including permutation test for assessing statistical significance.

Data available on NIMH archive [link](https://nda.nih.gov/study.html?id=587) or by request.
